Preguntas Frecuentes

  • ¿Qué puede subirse a Fotaza.net?

    La respuesta está en nuestros términos de uso.

  • ¿Cómo darle formato a los posts?

    Con la sintaxis Markdown, la guía está aquí.

  • ¿Qué tipos de archivos son admitidos?

    PNG, JPEG, BMP, y GIF, siempre y cuando no sea más grande que 10 MiB.

  • ¿Se admite subir videos?

    No. El espacio del disco duro es caro, y todo éste sitio está corriendo en una sola laptop. Pero sí es posible enlazar videos de otras páginas.

  • ¿Son los nombres únicos?

    Sí. Sólo puede haber una cuenta por nombre. Normalmente habrá un aviso si el nombre de usuario de la nueva cuenta está tomado.

  • ¿Cómo hago que mi post llegue a los Destacados?

    No es posible ayudarte con éso. Es necesario que la subida tenga al menos una calificación de 4/5 y 50 votos. Una vez conseguido éso, el post se quedará ahí hasta que haya pasado una semana desde que se subió.

  • ¿Es posible hacer un sitio similar?

    Sí. Fotaza.net es software libre bajo la GPL V3. A resumidas cuentas, es posible hacer un derivado siempre y cuando se redistribuya bajo los mismos términos.

  • ¿Es posible contribuir al código del sitio?

    Sí. Ve al repositorio de GitHub para contribuir o incluso hacer un trabajo parecido.

  • ¿Con qué está hecho éste sitio?

    Para el servidor, éste sitio usa: Node.js 18.18.2 y Express.js 4.18.2.
  • MariaDB para la base de datos.
  • Éste sitio corre en una pila LAMP/XAMPP.
  • Excepto ExpressJS, ninguno de los programas anteriores está incluido en el paquete.
    • El paquete incluye:
  • El cliente node-mysql2 de sidorares 3.5.1 para interactuar con la base de datos.
  • El ORM Sequelize 6.32.1 para interactuar con la base de datos.
  • BCrypt 5.1 para el "salteado" de contraseñas.
  • JSONWebToken de Auth0 9.0.1 para generar y firmar "cookies" estrictamente funcionales.
  • Cookie-parser de ExpressJS 1.4.6 para manejar las "cookies".
  • Node-formidable 3.5.0 para manejar los archivos entrantes.
  • Image-thumbnail de onildoaguiar 1.0.15 para generar miniaturas.
  • Jimp 0.22.10 para aplicar las marcas de agua.
  • Marked 10.0 para interpretar texto Markdown.
  • Sanitize-html de ApostropheCMS 2.11 para prevenir ataques XSS.
  • Pug 3.0.2 para presentar las páginas dinámicamente.
  • jQuery 3.7.1 para algo de lógica del lado del cliente.
  • Todos ésos programas excepto MariaDB están bajo la licencia MIT.
