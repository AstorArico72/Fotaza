Guía de HTML y formato

Ésto funciona para subidas, comentarios, y perfiles de usuario. En Fotaza, es necesario el marcado HTML para que el texto se vea bien, y no esté todo junto.
Encerrar el texto quiere decir poner la <etiqueta> delante de una porción de texto y una </etiqueta> detrás, ésa última con una barra delante, como se ve aquí.

  • Párrafos

    Encierra tu texto entre las etiquetas <p>. Para un salto de línea usa las etiquetas <br>.

  • Encabezados

    Encierra tu texto entre las etiquetas <h1>, <h2>... <h6>.

  • Negrita e itálica

    Encierra tu texto entre las etiquetas <strong> o <b> para el texto en negrita. Usa las etiquetas <em> para el texto en itálica.

  • Listas

    Encierra el texto entre las etiquetas <ol> o <ul>
    <ul> sirve para una lista de viñetas.
    <ol> sirve para una lista ordenada por números o letras.
    Añade el atributo "type" a la etiqueta que abre la lista (así: <ol type="a"> </ol>) para decidir si ordenar por números (type="1", type="i" o type="I" para números romanos) o letras (type="a" o type="A").
    Una vez hecho éso, encierra los ítems de la lista entre etiquetas <li>. Los ítems se ordenarán automáticamente según el orden en que estén escritos.
    Todo ésto se resume en:
    <ol type="a">
    <li> Item 1 </li>
    <li> Item 2 </li>
    <li> Item 3 </li>
    </ol>
    Puedes poner cualquier otra etiqueta dentro de una lista o ítem.

  • Enlaces externos

    Encierra el texto entre las etiquetas <a>, y usa el atributo "href" en la primera etiqueta para especificar una dirección, así: <a href="www.example.net">Enlace</a>
    También puedes añadir un título o leyenda al enlace añadiendo el atributo "title" junto al atributo "href", así: <a href="www.example.net" title="Ejemplo">Enlace</a>

  • Imágenes externas

    Usa la etiqueta <img> para colocar una imágen. No hay que poner una etiqueta para cerrarla.
    Hay que usar el atributo "src" de la misma forma que como se usa el "href" en <a> para especificar de dónde sacar la imágen. También se puede poner un título o leyenda a la imágen enlazada de la misma forma que con los <a>.
    Puedes especificar el ancho y alto usando los atributos "width" y "height" respectivamente, en píxeles (width="300px") o porcentaje (height="50%").
    Todo éso se resume en: <img src="www.example.net/example.png" height="100%" width="100%" title="Ejemplo">

  • Videos y audio externos

    Usa la etiqueta <video> de forma parecida a <img>. Puedes especificar el tamaño igual que en los <img>. La etiqueta <video> tiene que cerrarse, a diferencia de <img>, porque ahí dentro tiene que ir la etiqueta <source>, que debe llevar el atributo "src" dentro, igual que "src" en los <img>, debe enlazar al video.
    Puedes añadir atributos adicionales a la etiqueta <video> así: <video controls autoplay muted>, donde "autoplay" especifica reproducción automática, "muted" especifica reproducir inicialmente sin sonido, y "controls" especifica mostrar los controles del video. Ésos son sólo algunos ejemplos.
    Todo éso se resume en: <video width="1920px" height="1080px" autoplay> <source src="www.example.net/example.mp4"> </video>
    Es exactamente igual para el audio externo, sólo reemplaza "video" por "audio", y enlaza a algún archivo de audio en lugar de video.

  • Videos de YouTube y iframes

    Usa la etiqueta <iframe> para hacer un cuadro, en éste caso el reproductor de YouTube, y usa el atributo "src" igual que en las imágenes, sólo que no es necesario enlazar al archivo, sino a la página del video. Al igual que las imágenes, puedes especificar el tamaño, pero, a diferencia de éstas, las etiquetas deben cerrarse.
    Así: <iframe src="https://www.youtube.com/embed/(dirección)" width="1024" height="768"> </iframe>
    Ésto también sirve para embeber otros tipos de contenido en el post o comentario.

  • Estilo

    Revisa la hoja de estilos para ver qué clases puedes añadirle al texto, cosa que puede aplicarse a cualquier etiqueta, por ejemplo <a class="example">.
    También puedes añadir tus propias reglas CSS mediante el atributo "style" que, igual que "class", puede aplicarse a cualquier etiqueta.
